How to Embed Your Colab Notebook

Google Colab notebooks cannot be directly embedded on other websites due to security policies. However, you can publish your work in a few simple steps using GitHub, NBViewer and Binder. Follow the guide below to get your notebook ready for embedding:

  1. Publish your notebook to GitHub or Gist. In Colab, go to File > Save a copy to GitHub or File > Save a copy as Gist. Make sure the repository or Gist is public so others can access it.
  2. Generate a static HTML view. Use NBViewer to render the notebook. NBViewer takes a GitHub or Gist URL and produces a static HTML page. Copy the resulting NBViewer URL.
  3. Optionally create an interactive version. If you want visitors to run your code, you can use Binder with your GitHub repository. Binder launches a temporary Jupyter environment that runs your notebook. Use a service like NBInteract to generate embeddable HTML snippets that talk to Binder.
  4. Replace the iframe URL. Edit the src attribute of the iframe in the â€œFeatured Notebookâ€ section above with the NBViewer or NBInteract URL. Save your changes and refresh the page.
  5. Test your embed. Reload your website to confirm the notebook appears and functions as expected. Note that some services may still block embedding; if that happens, provide a link next to the iframe so visitors can open the notebook in a new tab instead.
